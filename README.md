# vue-timerapp

## Stack
Vue, Vuetify

![vuejs app timer](https://i.imgur.com/iCG326c.png)

## Features
1. Start timer,
2. Lap time. It will add current time into laps. Laps can be viewed in navigation bar
3. Pause timer
4. Continue timer after pause
5. Stop timer

## Project setup

```
npm install
```
